## Setup & Installation
### Docker
Build the images (PostgreSQL and App Image)
``` sh
docker-compose build
```

Make the containers for the images

``` sh
docker-compose up -d
```

See the logs

```
docker-compose logs -f
```
- Note: This step it's **OPTIONAL**
